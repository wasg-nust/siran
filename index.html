    Background

    Modsecurity provides a mechanism where rule on the fly can be fed into the firewall for the detection of latest threats. Creation of these rules is dependent on user experience and exposure; however; even subject to availability of ideal knowledge there is no guarantee that rules being fed are formally correct to detect latest threat for which they have been designed. It could end as futile effort or more towards false positive results. The main driving force for a firewall, Antivirus Solution or any other security product is its signature or mechanism through which it claims its effectiveness. If this seed has a flaw in its anatomy, fruitful results would always be awaited and will affect the cost of starvation of reputation, revenue and services as well.

    Siran

    Our solution to improve the usability of developing and deploying Web Application Firewall rules by allowing easy management, searching, analysis, translation and classification of firewall rules. It provides translation of ModSecurity rules to semantic representation and semantic rules to ModSecurity rules.

    Features of Siran

    We claim following novelties and state of the art where creation of rules is not only easy but can also be formally verified along with varieties of features for its management.

    Highly interactive GUI

    • Easy to use and understand
    • Novice and Non-technical person can even define security rules

    Semantic based analysis of rules

    • Rule Validation
    • Rule Inconsistency Checking
    • Rule Generalization: reducing search space

    Rule Explorer

    • Provide ability to construct effective rules
    • Easy Navigation
    • Keyword based semantic search in rule repository
    • Threat based and Priority based classifcation of Rules
